# Portfolio Website

Welcome to my portfolio website! This project showcases my work as a software developer, including several key projects that highlight my skills in front-end and back-end development. The portfolio is built using HTML, CSS, and JavaScript, and is hosted on [Glitch](https://glitch.com/).

## Features

- **Clean and Responsive Design**: The website is designed to be user-friendly and adaptable to different screen sizes, providing an optimal experience on both desktop and mobile devices.
- **Project Showcase**: Each project includes a brief description, technology stack, and links to live demos and GitHub repositories.
- **Contact Section**: Visitors can find links to connect with me on LinkedIn, view my GitHub profile, or send me an email.
